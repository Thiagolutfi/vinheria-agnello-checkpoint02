Projeto: Cadastro de Vinhos

Descrição do Projeto
Este projeto foi desenvolvido em JavaScript com o objetivo de realizar o cadastro de vinhos, permitindo armazenar informações como nome, ano da safra e quantidade em estoque.

O sistema faz validações de entrada, classificação automática dos vinhos e gera um relatório final com informações importantes sobre os cadastros realizados.



Funcionalidades Implementadas no Checkpoint 02

- Validação de texto para evitar campos vazios
- Validação numérica para garantir entradas válidas
- Verificação automática de estoque baixo
- Classificação dos vinhos por ano da safra:
  - Vinho Jovem
  - Vinho Amadurecido
  - Vinho Antigo
- Cadastro contínuo utilizando estrutura `while`
- Contador de total de vinhos cadastrados
- Contador de vinhos com estoque baixo
- Identificação do vinho mais antigo cadastrado
- Exibição de relatório final no console
